# Cursor at Windows

## 安装

1. 安装 Cursor

https://cursor.com/cn

2. 登录

可以使用 Github 账户登录

3. 安装Github 插件

```text
File -> Add Folder to Workspace...

编辑一个文件，比如：README.md

Publish to Github

配置 git：

  打开Git CMD
  git config --global user.name xxx
  git config --global user.emale xxx@xxx.com

Message：

  首次签入

Commit

Publish Branch "main"
```
